Chapter 3: Atoms, Molecules, and Ions

📘 SS1 First Term⏱️ Reading: 35 mins

🎯 Learning Objectives

  • Describe atomic structure
  • Define atomic number and mass number
  • Explain isotopes
  • Distinguish atoms, molecules, and ions
  • Write chemical formulas

3.1 Structure of the Atom

📌 Atom: The smallest particle of an element that can take part in a chemical reaction. It consists of protons, neutrons, and electrons.
ParticleSymbolChargeMassLocation
Protonp⁺+11Nucleus
Neutronn⁰01Nucleus
Electrone⁻−11/1840Shells (orbits)

3.2 Atomic Number and Mass Number

📌 Atomic Number (Z): Number of protons in an atom.
📌 Mass Number (A): Number of protons + neutrons.
A = Z + N   (N = number of neutrons)
📐 Example: Sodium (Na): Z=11, A=23. Neutrons = 23−11 = 12 neutrons.

3.3 Isotopes

📌 Isotopes: Atoms of the same element with the same number of protons but different numbers of neutrons.
📐 Examples:
• Carbon-12: 6p, 6n | Carbon-14: 6p, 8n
• Chlorine-35: 17p, 18n | Chlorine-37: 17p, 20n

Isotopes have the same chemical properties but different physical properties (mass).

3.4 Molecules

📌 Molecule: Two or more atoms chemically bonded together. Can be same element (O₂, H₂, Cl₂) or different elements (H₂O, CO₂).
Diatomic MoleculesPolyatomic Molecules
H₂, O₂, N₂, Cl₂, I₂, Br₂, F₂H₂O, CO₂, NH₃, CH₄, C₆H₁₂O₆

3.5 Ions

📌 Ion: A charged particle formed when an atom gains or loses electrons.
Cation (+)Anion (−)
Loses electronsGains electrons
Metals form cationsNon-metals form anions
Na⁺, K⁺, Ca²⁺, Al³⁺Cl⁻, O²⁻, SO₄²⁻, NO₃⁻

3.6 Writing Chemical Formulas

Rules:

  1. Write the symbols of elements
  2. Write their valencies (combining power)
  3. Cross-multiply valencies
  4. Simplify if possible
📐 Example: Aluminum oxide: Al³⁺ O²⁻ → Al₂O₃
📐 Example: Calcium chloride: Ca²⁺ Cl⁻ → CaCl₂
RadicalFormulaValency
HydroxideOH⁻1
NitrateNO₃⁻1
SulfateSO₄²⁻2
CarbonateCO₃²⁻2
AmmoniumNH₄⁺1

📝 Chapter 3 Summary

  • Atom = protons + neutrons + electrons
  • Atomic number (Z) = protons; Mass number (A) = protons + neutrons
  • Isotopes = same Z, different A
  • Molecules = 2+ atoms bonded
  • Ions = charged particles (cations +, anions −)
